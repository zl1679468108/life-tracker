开发标注与验收规格

布局

手机基准 390 x 844；本稿缩放为 300 x 642。页面左右边距 16，列表间距 8，底部预留 TabBar + 16。

字体

页面标题 22/30/700，区块 16/24/700，列表标题 13/18/700，说明 10-11/16。

触控

图标按钮、行、按钮的可点击区域均不小于 44 x 44。危险动作必须二次确认。
