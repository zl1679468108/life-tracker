# LifeTracker 任务计划

## 当前状态

**项目**: 生活管理应用 (物品/待办/分类/位置)
**技术栈**: Expo (React Native) + NestJS + Supabase
**进度**: 基础功能已完成，PWA 已配置

## 已完成

- 项目架构 (Monorepo)
- 前端 Expo + Web (PWA)
- 后端 NestJS + Supabase
- 用户认证 (登录/注册/重置密码)
- 物品/待办/分类/位置 CRUD
- 自建 UI 组件库
- PWA 配置
